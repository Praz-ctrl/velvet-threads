# Velvet Threads
## Luxury Clothing Store
Velvet Threads is a luxury clothing store with a minimalist and elegant aesthetic. Our mission is to provide high-quality clothing with a focus on comfort and style.
## Features
* Smooth scroll animations
* Parallax effect
* Floating cart icon
* Hamburger menu
* Responsive design
## Technologies Used
* HTML5
* CSS3
* JavaScript
* No frameworks or libraries used
## Getting Started
1. Clone the repository using `git clone https://github.com/username/velvet-threads.git`
2. Open the `index.html` file in a web browser
